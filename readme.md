# Разработка сервиса составления расписания для сети кофеен

## Описание проекта

Команды должны сами определить, для какого формата заведения разрабатывается сервис (кофейня, пекарня, ресторан) и какие роли участвуют в работе (управляющий, менеджер, бариста, стажёр).
Нужно продумать, как сотрудники подают пожелания по сменам, как управляющий их учитывает, какие сценарии возникают при замене смен, опозданиях и редактировании графика.
Следует предусмотреть хранение истории изменений, отображение актуальности расписания и сбор статистики по часам.
Важно показать связь между контекстом выбранной кофейни, реализованными функциями и архитектурными решениями — чтобы проект выглядел как реальный инструмент для управления персоналом, а не как абстрактное расписание.

## Цель
Создать цифровой инструмент для планирования и учёта рабочего времени, который уменьшает трудозатраты управляющих, повышает прозрачность для сотрудников и автоматически собирает статистику для HR и бухгалтерии.

## Результат (продукт)
Прототип сервиса с модулями:
1. составление и редактирование расписания
2. просмотр расписания сотрудниками и менеджерами
3. учёт часов и отработанного времени
4. уведомления об изменениях и замене смен
5. личный кабинет сотрудника с информацией о медосмотрах, аттестациях и статусе

## Критерии приемки
1. расписание создаётся без внешних таблиц и доступно всем ролям
2. бариста видит своё расписание и коллег по смене
3. управляющий может корректировать график и учитывать пожелания сотрудников
4. функциональность связана между собой и отражает реальные процессы кофейни
5. интерфейс адаптирован под мобильные устройства
